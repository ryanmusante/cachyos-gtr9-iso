  * Fix MED: chmod 755 on 3 files — 3 sequential calls; single chmod
  * Fix MED: _extract_fish_var had no argument validation; added count

  * Fix LOW: Step 4b grep '@@.*@@' error output leaked to stdout; added

  * Fix LOW: Summary section had orphaned blank lines on stdout between

3.7.0 (2026-03-09)
-------------------

  * Initial release
  * setup.fish: build-time ISO preparation script
    - Extracts configs from running GTR9 Pro via ry-install.fish
    - Injects @@KERNEL_PARAMS@@, @@MASK@@, @@PKGS_DEL@@ placeholders
    - Validates ry-install version match
    - Creates airootfs overlay structure
  * ry-install-post.sh: Calamares shellprocess hook
    - Generates /etc/kernel/cmdline with root UUID
    - Masks 9 services/targets
    - Enables 4 services
    - Removes 7 conflicting packages
    - Rebuilds initramfs and boot entries
  * ry-install-firstboot.sh: first-boot validation
    - Runs ry-install --verify-static
    - Self-disables via ConditionPathExists sentinel
  * ry-install-firstboot.service: systemd oneshot unit
  * shellprocess-ry-install.conf: Calamares module config
  * README.md: full implementation plan with phases, risks, maintenance
